Descrevendo algumas tags HTML* de formatação de texto.

A tag < em >

Está tag é ultilizada para dar ênfase ao texto contido em uma página HTML. O seu comportamento padrão exibe o texto em itálico.

A tag < strong >

Está tag é ultilizada para dar importância ao texto. O seu comportamento padrão exibe o texto em negrito.

A tag < sup > e < sub >

Estas tags são ultilizadas para definir seu texto sobescrito e subscrito, respectivamente. São ultilizadas em fórmulas químicas (H2SO4) e expressões matemáticas com potência (23=8), por exemplo.

A tag < ins > e < del >

Estas tags são ultilizadas para definir inserções de textos (com a tag < ins >) e remoções de texto (com a tag < del > ). Inserções e remoções são comuns em artigos de blogs, quando o autor quer deixar o usuário ciente do que houve uma atualização atualização no texto, seja ela uma correção, inserção ou inclusão de palavra(s). O comportamento padrão deixa o texto sublinhado para inserções e tachado para exclusões.

A tag < mark >

Está tag é ulilizada para marcar trechos do texto. É possível marcar tanto palavras específicas quando um conjunto de palavras. O comportamento padrão deixa o texto na cor preta e no fundo amarelo.
HTML* Hyper Text Markup Language
