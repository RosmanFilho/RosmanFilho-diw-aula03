Significado das tags HTML*

A tag < Address >

A tag address delimita semânticamente através das tags de abertura e fechamento, o endereço físico e eletrônico do autor de um documento HTML.
Exemplo:
Site Faculdade Promove Site
Endereço: Rua João Pinheiro
Telefone: 0800 031 2103

A tag < blockquote >

A tag blockquote indica que o texto incluído é uma longa citação.
Exemplo:
Esta é uma citação tirada da Mozilla Developer Center.

A tag < code >

A tag code serve para marcar o texto como sendo código de computador em uma linha ou frase.
Exemplo de um variável no php: $variável = $valor;

A tag < dfn >

A tag dfn possui tags de abertura e fechamento, contém um termo que é descrito no parágrafo onde está inserido.
Exemplo:

O termo e sua definição.

A tag < pre >

A tag pre é utilizada para representar um texto pré-formatado. Um texto dentro desse elemento é tipicamente exibido em uma fonte não proporcional da mesma maneira em que o texto original foi disposto no arquivo.
Exemplo: 
    A alegria evita mil males e prolonga a vida.

A tag < span >

A tag span é um conteiner generico em linha para conteúdo fraseado, que não representa nada por natureza. Ele pode ser usado para agrupar elementos para fins de outros estilos.
Exemplo:

A vida é Bela.

A tag < time >

A tag time representa o tempo tanto no formato de 24 horas ou como uma data precisa no calendário Gregoriano com informações opcionais de tempo e fuso horário.
Exemplo:

A aula está marcada para as 19:00.
